### System Requirements
- **Zotero 7** (Currently only supports Zotero 7, Zotero 8 support is under development)
- Windows / macOS / Linux

### Installation Steps

1. **Download Plugin**
   - Go to [Releases](https://github.com/BaiRuic/BibGenie/releases) page
   - Download the latest `.xpi` file

2. **Install to Zotero**
   - Open Zotero
   - Go to `Tools` â†’ `Add-ons`
   - Click the gear icon âš™ï¸ in the top right corner
   - Select `Install Add-on From File...`
   - Select the downloaded `.xpi` file
   - Restart Zotero

## â“ FAQ

<details>
<summary><strong>Do you offer a free trial?</strong></summary>
<br>
For free users, we offer permanent free usage rights. You only need to configure your own AI model API Key to use it.
</details>

<details>
<summary><strong>What document formats are supported?</strong></summary>
<br>
Currently, only PDF attachments are supported. Support for other file types is under development and will be available soon.
</details>

<details>
<summary><strong>How does Zotero Copilot securely access my research papers?</strong></summary>
<br>
Zotero Copilot connects to your Zotero library through official, secure APIs. Your research data remains completely private and secure, stored locally on your device. We never upload your research content or index to external servers.
</details>

<details>
<summary><strong>Can I integrate my own AI models and APIs?</strong></summary>
<br>
Absolutely! Zotero Copilot supports custom LLM configurations where you can set up multiple AI models using your own API Host and Keys. You can seamlessly switch between different models during research sessions for optimal results.
</details>

<br>
Currently, only Zotero 7 is supported. Zotero 8 support is under development and will be available soon.
